# 🚀 Taskly

## 🛠 Технологии

- **Backend:** Node.js + Express.js
- **Frontend:** HTML, EJS, Tailwind CSS
- **Database:** MongoDB (mongoose)
- **Tools:** Nodemon, PostCSS, Git

## 📁 Структура проекта

taskly/
├── backend/ # Backend часть
│ ├── config/ # Конфигурации
│ ├── controllers/ # Контроллеры
│ ├── models/ # Модели данных
│ ├── routes/ # Маршруты API
│ ├── services/ # Бизнес-логика
│ └── app.js # Точка входа сервера
├── frontend/ # Frontend часть
│ ├── assets/ # Исходные файлы (CSS, изображения)
│ ├── public/ # Скомпилированные статические файлы
│ ├── public/js/ # Клиентские скрипты
│ └── views/ # EJS шаблоны
├── .env # Переменные окружения
├── .gitignore # Git игнорируемые файлы
├── package.json # Зависимости и скрипты
├── tailwind.config.js # Конфигурация Tailwind
└── README.md # Документация
